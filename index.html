Home GitHub Repo Weather Data

The Weather Data website is a collection of 5,000 locations to visualize average temperature, humidity, windspeed, and cloudiness globally. Using Python, I dropped a random pin on the globe, and found the nearest city. Using the Underground Weather API, I pulled the weather data for that city. Once the data was cleaned by dropping nulls and making sure the data accurate, the results were plotted using Matplotlib Scatter Plots.

Earthquake Data

Have you ever wondered how many Earthquakes have occoured in the last 7 days? Well, now you can find out. Unfortunately, it is probably more than you think. At least, it is more than I thought. The diameter of the circle expresses the magnitude. This visualization used data from the United States Geological Survey (USGS) and is visualized using Plotly. Check out the Dark Map too.
